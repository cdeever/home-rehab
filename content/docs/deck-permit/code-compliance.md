## Lateral Load Path

Per IRC R507.9.2, at least two tension ties (e.g., Simpson DTT2Z) connect the deck joists to interior floor framing through the house rim to resist lateral loads — detail per Franklin County Deck Guide.

## Frost / Soil

- Frost depth: 32 in. below finished grade.
- Presumed soil bearing: 1,500 psf (per IRC Table R401.4.1, Class 4 — silt / sandy clay). Footings sized accordingly.
