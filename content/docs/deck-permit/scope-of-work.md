## Work To Be Performed

1. **Demolition.** Remove existing decking, railings, stairs, joists, beams, posts, and ledger. Dispose off-site.
2. **Footings.** Install new concrete pier footings, sized and spaced per the framing plan. Bear on undisturbed soil at or below local frost depth (32 in. below grade for Franklin County).
3. **Posts.** Set new pressure-treated 6×6 posts on footings using Simpson ABU66Z (or equivalent) standoff post bases.
4. **Beams.** Install multi-ply pressure-treated beams per framing plan; span and ply count per span tables in the Franklin County Deck Guide.
5. **Ledger.** Install new ledger attached to the house rim joist with through-bolts (½" diameter) and flashing per IRC R507.9; include drip cap / Z-flashing over ledger top.
6. **Joists.** Install pressure-treated joists at 16" o.c. (or tighter where span requires). Hang from ledger with Simpson LUS or equivalent joist hangers.
7. **Decking.** Install _TBD — pressure-treated 5/4×6 / composite board / etc._, fastened per manufacturer instructions.
8. **Railing.** Install code-compliant guardrail where walking surface is >30 in. above grade (36 in. min height, 4 in. sphere rule).
9. **Stairs.** _Include or remove based on design_. Rise / run to meet IRC R311.7.
10. **Lighting / electrical.** _None in this scope_ / _existing exterior receptacle remains_ — confirm.
